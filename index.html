About 3M

3M is a Python-based web application designed to simplify and secure the exam creation process for instructors through intelligent automation.

Built using the Flask framework, the system enables instructors to upload questions, answer choices, and correct answers through structured text files. Based on instructor-defined options, 3M automatically generates multiple exam paper groups by shuffling both the question order and answer choices for each group.

For every generated exam version, a corresponding answer sheet is produced to ensure accuracy and efficiency during grading. The platform also provides clear documentation for all inputs, outputs, templates, and internal processes, making it easy to use and maintain.

Our goal is to reduce manual effort, minimize cheating opportunities, and provide educators with a reliable, fair, and efficient exam generation system.
